## 元圭与 VCC 1.0.2

本次更新集中完善音乐陪伴体验，并加入应用内更新能力。

### 新功能

- 新增统一音乐播放器，可控制 Apple Music，并播放哔哩哔哩公开视频音频。
- 哔哩哔哩支持 URL、BV 号和短链接导入，以及播放列表、收藏、本地歌单和多种播放模式。
- 新增哔哩哔哩扫码登录；登录后可以读取账号有权访问的播放器字幕。
- 新增当前歌曲封面、桌面歌词样式、歌词偏移和手动 LRC 导入。
- 设置中新增“关于与更新”，可读取 GitHub Release 更新日志并一键下载安装新版本。

### 优化与修复

- 改进 LRCLIB 歌词匹配，支持歌曲名与歌手字段写反的情况，并加入超时与错误提示。
- 歌词、哔哩哔哩 URL、收藏、歌单和播放进度会持久保存，切换播放源或重启后不再丢失。
- 哔哩哔哩字幕增加登录态播放器接口、WBI 接口及 `subtitle_url_v2` 回退。
- 优化完整播放器和状态栏音乐卡片布局，播放列表限制为最多展示 4 首。
- 播放音乐时，元圭、VCC 和双人模式会自动切换到听歌动作。

### 系统要求

- macOS 14 Sonoma 或更高版本。
- 当前 DMG 为临时签名个人分享版；首次打开可能需要按住 Control 点击应用并选择“打开”。
