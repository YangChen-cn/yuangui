# 元圭与 VCC 1.0.4

本版本集中修复音乐播放器的字幕与歌词持久化问题，并改善歌曲信息编辑、歌词偏移和桌宠气泡的使用体验。

## 音乐与字幕

- 修复 Bilibili 播放器接口偶发保留正确 BV/CID、却返回其他视频字幕轨的问题。
- AI 字幕会校验 URL 中的 `aid` 与 `cid`；人工字幕会通过重复轨道 ID 确认，避免随机响应覆盖正确字幕。
- 已缓存的错误 Bilibili 字幕会在解析到正确字幕地址后自动失效并重新下载。
- Apple Music 的 LRCLIB 歌词使用稳定缓存键，忽略时长微小变化、大小写及空格差异，重启后无需重复匹配。
- 调整启动顺序，先恢复本地音乐资料库，再同步 Apple Music。

## 操作体验

- Bilibili 登录入口完整显示昵称与 UID，长昵称可显示两行。
- 新增“仅保存歌曲信息”，歌曲名和歌手可以单独修改，不触发 LRCLIB，也不会删除现有歌词。
- 歌词偏移支持直接输入秒数和 0.1 秒步进微调，并统一状态栏、完整播放器和歌词设置弹窗的操作方式。
- 提高 AI 回复与日常聊天气泡的不透明度，并使用自适应背景色改善深色模式和深色桌面的文字对比度。
